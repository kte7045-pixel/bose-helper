BOSE 도우미 서비스 가이드

개요

BOSE 제품의 문제 해결 및 AS 안내를 위한 대화형 웹 애플리케이션입니다.

주요 기능

1. 제품 카테고리

  • 헤드폰: QC Ultra 2세대, QC Ultra, QC, QC45, QC35, QC25
  • 이어버드: QC Ultra 이어버드 2세대, QC Ultra 이어버드, 울트라 오픈 이어버드, QC 이어버드 II
  • 스피커: 홈 스피커 시리즈, SoundLink Flex/Max/Revolve/Mini
  • 홈시어터: Soundbar Ultra/900/700/600/500/300, TV 스피커
  • 홈오디오: Wave SoundTouch, SoundTouch 10/20/30 (단종 제품 포함)

2. 문제 해결 흐름

시작 → 제품 선택 → 증상 선택 → 공장 초기화 안내 → 성공/실패 → AS 접수

QC Ultra 헤드폰 2세대 매뉴얼

배터리 사용 시간

  • ANC 켜짐 + 몰입형 오디오 꺼짐: 최대 30시간
  • ANC 켜짐 + 몰입형 오디오 켜짐: 최대 23시간
  • ANC 꺼짐: 최대 45시간
  • 급속 충전: 15분 충전으로 최대 3시간 사용

버튼 조작법

블루투스/전원 버튼 (오른쪽 이어컵 뒷면)

  • 짧게 누름: 전원 켜기
  • 길게 누름: 전원 끄기
  • 계속 누름: 페어링 모드

다기능 버튼 (오른쪽 이어컵 아랫면)

  • 1번: 재생/일시정지, 전화 받기/끊기
  • 2번: 다음 트랙
  • 3번: 이전 트랙
  • 길게: 청취 모드 순환

볼륨 스트립 (오른쪽 이어컵 뒷면 세로)

  • 위로 쓸기: 볼륨 올리기
  • 아래로 쓸기: 볼륨 내리기
  • 길게 터치: 단축키 실행

청취 모드

  • Quiet 모드: 완전한 노이즈 캔슬링
  • Aware 모드: 투명 모드 (주변음 허용)
  • Immersion 모드: 공간음향 (음악 최적화)
  • Cinema 모드: 공간음향 (영상 최적화)

블루투스 연결

처음 연결하기

  1. Bose 앱 설치 후 전원 켜면 자동 감지
  2. 수동: 블루투스 버튼 길게 → 파란 불 깜빡이면 연결
  3. Fast Pair (Android): 페어링 모드에서 자동 팝업

멀티포인트

공장 초기화 방법

헤드폰 제품

QC Ultra 헤드폰 2세대 / QC Ultra 헤드폰

  1. Bose 앱 제거
  2. 블루투스 목록 삭제
  3. 블루투스/전원 버튼 + 다기능 버튼 동시에 15초 누름
  4. 흰색 불 → 완료

QC 헤드폰

  1. Bose 앱 제거
  2. 슬라이더를 BT 방향으로 밀어 페어링 모드 진입
  3. ANC 버튼 10초 누름
  4. 표시등 흰색 깜빡 → 완료

QC45 헤드폰

  1. Bose 앱 제거
  2. 볼륨 − + 다기능 버튼 동시에 10초 누름
  3. 표시등 흰색 깜빡 → 완료

이어버드 제품

QC Ultra 이어버드 2세대

주의: 3번 반복 필수

  1. 앱 제거 & BT 삭제
  2. 케이스 넣고 덮개 오픈 → 뒷면 버튼 25초 누름
  3. 흰색 2번 깜빡 → 파란색 → 꺼짐
  4. 이 과정을 3번 반복
  5. 3번째 후 주황색 깜빡 → 파란색 천천히 깜빡 → 완료

QC 이어버드 II

1회만 실행

  1. 앱 제거 & 충전 케이블 연결
  2. 케이스 넣고 덮개 오픈 → 뒷면 버튼 30초 누름
  3. LED 흰색 고정 → 파란색 천천히 깜빡 → 완료

스피커 제품

홈 스피커 300/450/500

  1. Bose 앱 제거
  2. 볼륨 − + AUX 버튼 동시에 10초
  3. 라이트 바 2번 깜빡 → 완료

SoundLink Flex / Max

  1. Bose 앱 제거
  2. 전원 버튼 + 볼륨 − 버튼 동시에 10초
  3. 흰색 깜빡 → 재부팅 → 완료

블루투스 연결 문제 해결

새 기기에 연결이 안 될 때

방법 1: Bose 앱 (권장)

  • Bose 앱 설치 후 헤드폰 전원 켜기
  • 앱이 자동으로 감지하여 연결 안내

방법 2: 수동 페어링

  1. 헤드폰 전원 끄기
  2. 블루투스/전원 버튼 길게 눌러 파란 불 깜빡임 (페어링 모드)
  3. 스마트폰 블루투스 설정에서 BOSE QC ULTRA 2 HP 선택

방법 3: Fast Pair (Android 전용)

  • 페어링 모드 상태에서 Android 기기 가까이 두기
  • 화면에 팝업 알림 표시 → 탭하여 연결

쓰던 기기에 갑자기 안 될 때

  1. 기기 블루투스 껐다 켜기: 스마트폰 블루투스 재시작
  2. 헤드폰 전원 껐다 켜기: 헤드폰 완전히 껐다가 다시 켜기
  3. 블루투스 목록에서 삭제 후 재연결: 처음부터 다시 페어링

연결이 자꾸 끊길 때

원인 1: 거리 또는 장애물

  • 블루투스 안정 범위: 약 10m 이내
  • 벽, 사람 등 장애물 많으면 신호 약화

원인 2: 무선 간섭

  • 와이파이 공유기, 전자레인지 근처에서 끊김 심화
  • 위치 변경 테스트 권장

원인 3: 여러 기기 동시 연결 시도

  • 이전 연결 기기가 자동 연결 시도 시 끊김 가능
  • 사용하지 않는 기기의 블루투스 끄기
